# xq-server

## clone project

```
git clone https://github.com/xQuotes/xq-server.git

cd xq-server
```

## Running

Install dependencies
```
npm install
```

### devlopment

```
npm run dev
```


### Build

```
npm run build
```

### Start

```
npm start
```

### Browse

Open your browser, you can visit `http://localhost:3000`
